# 用户使用手册

> 工程质量安全智慧管理平台 | v1.0.0 | 适用于：水利工地监管人员、项目经理、安全员

---

## 1. 系统概述

### 1.1 系统简介

**工程质量安全智慧管理平台**是一套面向水利建设工地的智能化监管系统，对工地实现"全面感知、实时监测、智能预警、闭环处置"。

### 1.2 功能模块

| 模块 | 说明 | 入口 |
|------|------|------|
| **地图沙盘** | 基于 CesiumJS 的三维工地地图，支持传感器数据叠加、视频监控点位标注、无人机航线展示 | 前端：sandbox |
| **视频监控** | 实时视频流查看、AI 事件检测回放、多画面轮巡 | 前端：sandbox |
| **AI 检测** | 安全帽检测、越界检测、烟火识别、人流统计等 | 集成在视频监控中 |
| **告警管理** | 告警生成 → 推送 → 处置 → 归档 全流程闭环 | 前端：workflow |
| **专家系统** | 基于 RAG 的知识问答、自然语言生成数据表格 | 前端：expert |

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器，访问：`http://服务器IP/` 或 `https://your-domain.com/`
2. 输入用户名和密码（初始账号由管理员创建）
3. 点击"登录"，进入系统主界面
4. 系统默认显示**地图沙盘**页面

> 首次登录建议修改默认密码。

### 2.2 查看工地地图

1. 登录后自动进入地图沙盘（CesiumJS 三维场景）
2. **鼠标操作**：
   - 左键拖拽：旋转视角
   - 右键拖拽：平移地图
   - 滚轮：缩放
   - 左键单击：选中标注点
3. **右上角工具栏**：
   - 🌡️ 传感器图层开关（叠加温湿度、压力等数据）
   - 📹 摄像头图层开关（显示监控点位）
   - 🚁 无人机航线图层开关
   - 📐 测量工具（面积、距离量测）
4. **左下角图例**：显示各传感器类型颜色含义

### 2.3 查看实时监控

1. 在地图上点击 **摄像头图标**，弹出监控列表
2. 选择摄像头，右侧滑出实时视频画面
3. 支持操作：
   - ▶️ 播放 / ⏸️ 暂停
   - 🔊 开关声音
   - 📷 截图
   - 🎬 视频回放（点击时间轴选择时间段）
   - ⬜ 全屏

> 视频流基于 RTMP/HTTP-FLV 协议，加载需要 2-3 秒。

### 2.4 处理告警

1. 点击顶部导航栏 **告警管理**
2. 页面展示所有告警列表，支持筛选：
   - 告警级别：🔴 严重 / 🟠 重要 / 🟡 一般
   - 告警状态：待处理 / 处理中 / 已关闭
   - 时间范围
3. 点击某条告警，进入**告警详情**
4. 选择处置动作：确认为真实告警 / 误报 / 无效
5. 填写处理说明，点击**提交处置**
6. 告警状态更新，系统记录处置日志

## 3. 详细功能

### 3.1 地图沙盘（Sandbox）

#### 3.1.1 工地场景管理

- **切换工地**：下拉选择不同工地，显示对应地图和传感器
- **视角重置**：点击"回到初始视角"按钮，恢复默认视角
- **收藏视角**：右键保存当前视角，后续快速切换

#### 3.1.2 传感器数据叠加

开启传感器图层后，地图上显示传感器标注点，点击可查看：
- 当前数值
- 24 小时历史曲线图
- 告警阈值（红色警告线）

#### 3.1.3 标注与测量

- **添加标注**：在地图上右键 → "添加标注"，输入名称和描述
- **距离测量**：点击测量工具 → 依次点击起点和终点，显示距离
- **面积测量**：点击测量工具 → 依次点击多边形顶点，显示面积

#### 3.1.4 无人机航线

- 显示预设的无人机巡检航线（蓝色线条）
- 点击航线，显示航点列表和巡检计划
- 支持查看历史航次录像

### 3.2 视频监控（Sandbox）

#### 3.2.1 摄像头管理

- **添加摄像头**：管理员在 ThingsBoard 设备管理中添加摄像头设备
- **编组管理**：将摄像头按区域/功能编组，便于批量操作
- **云台控制**：支持 PTZ（云台）摄像头的转动和变焦

#### 3.2.2 多画面轮巡

- 在视频监控页面点击"**轮巡**"按钮
- 选择参与轮巡的摄像头列表
- 设置轮巡间隔（默认 10 秒）
- 画面自动按序切换

#### 3.2.3 AI 事件标注

- AI 检测到的安全事件在视频画面上叠加标注框
- 标注框颜色对应告警级别
- 点击标注框可查看事件详情并快速处置

### 3.3 告警管理（Workflow）

#### 3.3.1 告警创建

告警来源：
- **传感器告警**：数据超过阈值（如温度 > 60℃）
- **AI 检测告警**：安全帽未佩戴、越界、烟火等
- **人工上报**：用户手动创建告警

#### 3.3.2 告警推送

| 推送渠道 | 说明 | 配置 |
|----------|------|------|
| Web 界面 | 实时刷新，WebSocket 推送 | 默认开启 |
| 飞书群机器人 | 告警推送到预设飞书群 | 管理员配置 Webhook URL |
| 短信通知 | 重大告警短信通知责任人 | 管理员配置短信网关 |
| 邮件通知 | 告警摘要邮件发送 | 管理员配置邮件服务 |

#### 3.3.3 告警统计

- **实时大屏**：大屏展示今日告警统计（各级别告警数量、处理率）
- **报表导出**：支持导出 CSV/Excel 格式告警记录

### 3.4 专家系统（Expert）

#### 3.4.1 知识问答（RAG）

1. 在专家系统页面输入自然语言问题
2. 系统从知识库中检索相关文档片段
3. 生成答案并标注信息来源
4. 支持追问和上下文延续

**示例问题**：
- "大坝渗漏的应急处置流程是什么？"
- "高边坡施工有哪些安全规范？"
- "混凝土浇筑温度控制标准是多少？"

#### 3.4.2 表格生成

1. 输入数据查询请求（如："生成近一个月各工点的告警统计表"）
2. AI 自动从数据库查询数据
3. 生成结构化表格（支持导出为 Excel）

#### 3.4.3 知识库管理

管理员可上传和管理知识库文档：
- 支持格式：PDF、Word、TXT、Markdown
- 自动分片、向量化存储
- 支持增量更新

## 6. 常见问题

### Q1: 登录提示"用户名或密码错误"

- 检查是否输入了正确的用户名（区分大小写）
- 联系管理员重置密码
- 检查是否输入了正确的 URL（区分 http/https）

### Q2: 视频画面加载失败

1. 检查是否为支持的浏览器（Chrome 90+ / Edge 90+ / Firefox 90+）
2. 检查网络是否允许 1935 端口（RTMP）和 8087 端口（HTTP-FLV）
3. 刷新页面重试
4. 检查 EasyDarwin 流媒体服务是否正常运行：`curl http://localhost:10000/api/v1/stream/list`

### Q3: 传感器数据不更新

1. 检查设备是否在线（ThingsBoard 设备列表查看状态）
2. 检查 MQTT 消息是否正常上报
3. 查看 sensor-collector 日志：`docker compose logs sensor-collector`
4. 确认设备令牌未过期

### Q4: 告警未收到推送

1. 检查飞书群机器人 Webhook 是否配置正确
2. 检查告警级别是否满足推送条件（一般告警可能未配置推送）
3. 在告警管理页面手动触发一次告警，验证推送链路

### Q5: 知识库问答返回"未找到相关内容"

1. 确认知识库已上传并完成向量化
2. 尝试换一种问法（使用更具体的关键词）
3. 联系管理员检查知识库配置和 Milvus 服务状态

### Q6: 地图加载缓慢或卡顿

1. 检查网络带宽（CesiumJS 地图需要下载地形和影像数据）
2. 关闭不需要的图层（如多个传感器图层同时开启）
3. 浏览器硬件加速是否开启：`chrome://settings/system`
4. 尝试刷新页面或更换浏览器

### Q7: 如何导出告警数据？

1. 进入"告警管理"页面
2. 设置筛选条件（时间范围、级别、状态）
3. 点击右上角"**导出**"按钮
4. 选择格式（CSV / Excel），下载文件
